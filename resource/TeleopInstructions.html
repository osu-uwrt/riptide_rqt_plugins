Controls:

Start: Enable Controller (And dive if at surface)

X: Disable and stop controlller

Square: Disable but hold position

Circle: Zero pitch and roll


Right Stick:

Up/Down: Forward/Backwards

Left/Right: Turn


Left Stick:

Up/Down: Elevation

Left/Right: Strafe Left/Right


D-Pad:

Up/Down: Pitch

Left/Right: Roll


Note:

A PS3 controller has been tested to be used with this. If another controller does not work, check the log to ensure that it has all of the axes expected in the ps3_teleop node.


If the controller is not working, check the console for errors. If the joystick node is running the rqt log should contain logging that it opened the joystick (with the name of the joystick). If the teleop controller is publishing position, it should appear as published in this plugin. If any issues occur while reading the joystick (such as the joystick not matching the right format) they should appear in the ros bringup log as errors.
